Spamikaze is an automated spam blocklist system, designed to:

  • block spam at the SMTP level
  • reduce false positives
  • work with existing mail servers
  • make sending spam as annoying as receiving spam

Unlike some other spam blocking systems, Spamikaze does no tests for open relay or open proxy vulnerabilities at all. Instead, Spamikaze simply lists the IP addresses that have sent spam and allows anybody to remove IP addresses from the list. Next to the mta version there is also a version using a nameserver in order to enable your own RBL.

Spamikaze is designed to work on any unixtm based OS.
